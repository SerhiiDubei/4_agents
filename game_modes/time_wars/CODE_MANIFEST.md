# TIME WARS: Маніфест Кодів v1.0

## 1. Базова одиниця

**1 хв (власного часу) = 20 мани = X**

Це якір. Від нього рахується все.

---

## 2. Типи хвилин та їх коефіцієнти value

| Тип часу | Код | Опис | Коефіцієнт (k) | Пояснення |
|----------|-----|------|----------------|------------|
| A — Собі | `self` | Додаєш час собі | 1.0 | Базове value |
| B — Забрати | `steal` | Переносиш від суперника до себе | 1.3 | Краще, бо й собі +, і ворогу − |
| C — Другу | `give` | Додаєш час іншому (не собі) | 0.5 | Гірше, бо не тобі |
| D — Мінус всім | `minus_all` | Забираєш у всіх, включно з собою | 0.3 | Дуже ризиковано, мало value |
| E — Плюс всім крім одного | `plus_all_except_one` | Додаєш усім, крім цілі | 0.8 | Соціальний ефект, але не фокус |
| F — Нуль-сум | `zero_sum` | Перекид між гравцями | 1.1 | Трохи краще за A, бо агресія |

## 3. Класи кодів (c, b, a, S)

| Клас | Діапазон цін (мана) | Коеф value | Призначення |
|------|---------------------|------------|-------------|
| c | 6–16 (0.3X–0.8X) | 0.5 | Budget — дешево, для початку |
| b | 20–30 (1.0X–1.5X) | 0.75 | Standard |
| a | 40–60 (2.0X–3.0X) | 1.0 | Premium |
| S | 60+ (3.0X+) | 1.25 | Comeback / Top |

---

## 4. Формула вартості коду

```
Вартість (мана) = (Base_EV × k × Risk_Multiplier × Position_Modifier) × X
```

де:

- **Base_EV** = очікувані хвилини ефекту
- **k** = коефіцієнт типу часу (з таблиці вище)
- **Risk_Multiplier** від risk_level: 0 → 0.7 (стабільний), 1 → 1.0 (ризик), 2 → 1.5 (хаос)
- **Position_Modifier** = 1.0 (перші місця) / 0.7 (останні місця, знижка)

---

## 5. Принцип ескалації

| Сегмент | Діапазон цін | EV за 1 ману | Призначення |
|---------|--------------|--------------|-------------|
| Budget | 6–16 | 0.03–0.04 хв | Дешево, неефективно, для початку |
| Standard | 20–30 | 0.05–0.07 хв | Оптимальний баланс |
| Premium | 40–60 | 0.08–0.12 хв | Дорого, супер-ефективно |
| Comeback | 60+ | 0.15+ хв | Тільки для останніх місць |

---

## 6. Приклад: код Vampire

Забираєш 2 хв у суперника, даєш собі 1.5 хв (0.5 втрачається).

- Base_EV = 1.5 (що отримуєш собі)
- k = 1.3 (тип B — steal)
- Risk = 1.0 (стабільний)
- Position = 1.0

**Вартість** = (1.5 × 1.3 × 1.0 × 1.0) × 20 = **39 мани** → округлюємо до **40**

---

## 7. Зворотній розрахунок (Base_EV з ціни)

```
Base_EV = Вартість / (k × Risk_Multiplier × Position_Modifier × X)
```

---

## 8. Картка коду (повна атрибутика)

Усі поля присутні; якщо не застосовується — 0 або false.

- id, class (c|b|a|S), type, base_ev, cost_mana
- risk: bool, risk_level: 0|1|2
- position_gated: bool, position_modifier: float
- target_self, target_other, target_all, target_all_except_one: bool
- effect_self_min/max, effect_other_min/max: float
- choices: list — кожен елемент: id, name?, target_self/other, outcomes: [{ effect_self, effect_other?, probability }]

Якщо кілька choices — гравець обирає одну опцію; потім розігрується один outcome за probability.
