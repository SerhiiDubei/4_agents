POST /submit-answer/{session_id}
Request Body
{ "question_id": "q_0001", "answer": "survive", "free_text": "optional elaboration" }
Response
{ "core_delta": { "fear": +0.05, "aggression": +0.02 }, "questions_remaining": 7, "core_snapshot": { /* full CORE state */ } }
Submits an answer and returns the resulting CORE personality delta applied to the agent.

Response
{ "game_id": "game-uuid", "rounds": [ { "round": 1, "decisions": { /* agent → choice */ }, "payoffs": { /* agent → score */ } } ], "final_scores": { /* agent → total */ }, "dialog_log": [/* message objects */] }
Full round-by-round results for a single game, including dialog log, decisions, and payoff history.

GET /api/game-events/{session_id}
SSE Stream — Event Shape
// Content-Type: text/event-stream // Each event: data: { "type": "steal_attempt", "actor": "agent_A", "target": "human1", "time_delta": -25, "outcome": "SUCCESS", "roll": 17, "tick": 42 }
Server-Sent Events stream. Client connects once and receives tick events in real-time until game ends.
POST /api/human-action/{session_id}
Request Body
{ "player_id": "human1", "action": "STEAL", // or COOPERATE / PASS "target_id": "agent_A", // required for STEAL "card_id": "card_shield_A" // optional }
Response
{ "accepted": true, "queued_for_tick": 43, "your_time_remaining": 187 }
Human player submits their action for the current tick. Queued and resolved with all other actions at tick boundary.
GET /api/session-state/{session_id}

Request Body
{ "reason": "admin_stop" // or "timeout" }
