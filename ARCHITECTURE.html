# Кооперація: COOP_REWARD_EACH = 30 # +30s обом COOP_MANA_EACH = 5 # +5 мани обом COOP_TRUST_DELTA = 0.2 # trust+0.2 # Steal (d20 roll): STEAL_SUCCESS_THRESHOLD = 14 # 14+ = success STEAL_PARTIAL_THRESHOLD = 7 # 7-13 = partial STEAL_SUCCESS_TIME = 45 # -45s від жертви STEAL_ATTACKER_GAIN = 30 # +30s атакуючому # Drain (per tick): BASE_DRAIN_RATE = 1.0 # s/tick базово ESCALATION_FACTOR = 1.1 # +10% кожні N тіків
Баланс визначає динаміку гри. При поточних константах steal має EV>0 тільки проти слабких гравців з низьким trust.
shop.py / code_manifest.py — Коди
# Класи кодів: Клас C: дешеві, передбачувані - TIME_BOOST: +20s собі (ціна: 15 mana) - SHIELD: -50% steal наступний раунд Клас B: середні, з ризиком - DOUBLE_STEAL: steal x2 але якщо fail -20s - ALLIANCE: +trust з усіма +0.15 Клас A: потужні, дорогі - TIME_BOMB: таргет -60s (ціна: 40 mana) Клас S: рідкісні, ігрозмінні - CHRONO_RESET: всі гравці +120s
Коди купуються в shop через mana. Агент може тримати до N кодів. Використовуються в run_code_phase() тіку.
