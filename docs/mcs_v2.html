Raw scores — це "голоси" персон. Але вони ще не нормалізовані. Сума 2.10 + 1.40 + 0.90 + 0.60 = 5.00. Наступний крок — softmax.
softmax() — перетворення raw scores у ймовірності
weight[i] = e^raw[i] / Σ(e^raw[j])  →  сума всіх weights = 1.0
🛡️ Protector 2.10 → e²·¹⁰ = 8.17 ÷ 16.5 = 0.42
⚡ Instinct 1.40 → e¹·⁴⁰ = 4.05 ÷ 16.5 = 0.28
🧠 Thinker 0.90 → e⁰·⁹⁰ = 2.46 ÷ 16.5 = 0.18
🎭 Mask 0.60 → e⁰·⁶⁰ = 1.82 ÷ 16.5 = 0.12
Σ = 0.42 + 0.28 + 0.18 + 0.12 = 1.00 ✓
Softmax важливий: невелика різниця в raw scores → більша різниця у вагах. Протектор з 2.10 не просто "найсильніший" — він домінує у 3.5 рази над Маскою.
